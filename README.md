## 🧪 How to Run

1. Clone the repository
2. Configure MySQL database
3. Update application.properties
4. Run the project using:
   mvn spring-boot:run

## 📌 Sample Request

POST /jobs

{
"companyName": "TCS",
"role": "Java Developer",
"status": "Applied"
}
